# AppAgenda (Sistema de Agenda em Java)

## Visão Geral

O **AppAgenda** é uma aplicação desktop desenvolvida em Java com foco no gerenciamento de contatos e reuniões.  
O projeto segue uma arquitetura organizada em camadas, facilitando a manutenção e evolução do sistema.

---

## Estrutura do Projeto

```

AppAgenda-Java-main/
├── AppAgenda/
├── out/
│   └── production/
│       └── AppAgenda/
├── src/
│   ├── controller/
│   ├── model/
│   └── view/
├── AppAgenda.iml
└── Agenda-Java.iml

```id="agenda_tree_simple"

---

## Organização do Sistema

### src/controller
Contém as classes responsáveis pelas regras de negócio e controle das entidades do sistema.

---

### src/model
Responsável pela camada de acesso a dados e estruturação das informações do sistema.

---

### src/view
Contém a parte de execução e interação com o sistema, incluindo a classe principal.

---

## Pasta de Compilação

### out/production/AppAgenda
Contém os arquivos compilados (`.class`) gerados automaticamente durante a execução do projeto na IDE.

---

## Arquivos do Projeto

- **AppAgenda.iml**: configuração do módulo do IntelliJ IDEA  
- **Agenda-Java.iml**: configuração adicional do projeto  

---

## Funcionalidades

- Gerenciamento de contatos  
- Organização de reuniões  
- Estruturação de listas e informações  
- Separação em camadas (controller, model e view)  
- Aplicação orientada a objetos  

---

## Tecnologias Utilizadas

- Java  
- Programação Orientada a Objetos (POO)  
- Estrutura MVC simplificada  
- IntelliJ IDEA  

---

## Objetivo do Projeto

O objetivo do **AppAgenda** é praticar conceitos de:

- Organização de projetos Java  
- Estrutura em camadas  
- Separação de responsabilidades  
- Desenvolvimento de aplicações desktop
